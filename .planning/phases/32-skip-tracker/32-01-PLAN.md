Create `supabase/migrations/00016_skip_tracker.sql` containing a single `CREATE OR REPLACE FUNCTION` for `public.get_weekly_skip_counts`.

Function signature:
```sql
CREATE OR REPLACE FUNCTION public.get_weekly_skip_counts(
  p_student_ids  uuid[],
  p_today        date,
  p_current_hour integer
)
RETURNS jsonb
LANGUAGE plpgsql
STABLE
SECURITY DEFINER
SET search_path = public
```

Implementation details (per D-01, D-02, D-03 from CONTEXT.md):
1. Compute `v_week_start` as `date_trunc('week', p_today)::date` — this returns the Monday of the ISO week (D-03 reset behavior).
2. Compute `v_count_through`:
   - If `p_current_hour >= 23` then `v_count_through := p_today` (today counts as skippable per D-02)
   - Else `v_count_through := p_today - interval '1 day'` (today excluded per D-02)
3. Early exit: if `v_count_through < v_week_start`, return `'{}'::jsonb` (no countable days yet — e.g., Monday before 23:00 UTC).
4. For each student ID in `p_student_ids`, count skip days using:
   ```sql
   SELECT count(*) INTO v_skip_count
   FROM generate_series(v_week_start, v_count_through, interval '1 day') AS d(day_date)
   WHERE NOT EXISTS (
     SELECT 1 FROM work_sessions ws
     WHERE ws.student_id = v_sid
       AND ws.date = d.day_date::date
       AND ws.status = 'completed'
   )
   AND NOT EXISTS (
     SELECT 1 FROM daily_reports dr
     WHERE dr.student_id = v_sid
       AND dr.date = d.day_date::date
       AND dr.submitted_at IS NOT NULL
   );
   ```
5. Build the result JSONB: `v_result := v_result || jsonb_build_object(v_sid::text, v_skip_count)` for each student.
6. Return `v_result`.

CRITICAL correctness invariants:
- NEVER use `CURRENT_DATE` or `now()` anywhere in the function body (per SKIP-05, pitfall #1 in RESEARCH.md)
- `work_sessions` filter MUST include `AND ws.status = 'completed'` (pitfall #4)
- `daily_reports` filter MUST include `AND dr.submitted_at IS NOT NULL` (pitfall #5)
- Use `FOREACH v_sid IN ARRAY p_student_ids LOOP` for iteration (established pattern from 00011)
- Initialize `v_result` as `'{}'::jsonb`
