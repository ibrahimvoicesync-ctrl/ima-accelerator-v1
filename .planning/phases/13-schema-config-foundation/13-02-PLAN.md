Add the `getTodayUTC()` function to `src/lib/utils.ts` immediately AFTER the existing `getToday()` function (after line 15):

```typescript
/** Returns today's date as YYYY-MM-DD in UTC */
export function getTodayUTC(): string {
  return new Date().toISOString().split("T")[0];
}
```

This is a one-liner using `new Date().toISOString()` which always returns UTC. The `split("T")[0]` extracts just the date portion. This function is needed for all deadline arithmetic in Phases 14-18 so that server-side and client-side date math agrees regardless of user timezone.

Do NOT modify `getToday()` — both functions coexist. `getToday()` uses local time, `getTodayUTC()` uses UTC. Downstream code will use whichever is appropriate.
