**Roadmap step display function (per D-05):**

```typescript
function getStepDisplay(stepNumber: number | null): string {
  if (stepNumber === null) return "No active step";
  const step = ROADMAP_STEPS.find((s) => s.step === stepNumber);
  if (!step) return `Step ${stepNumber}`;
  return `Stage ${step.stage}: ${step.stageName} — ${step.title}`;
}
```

Example output: `"Stage 2: Influencer Outreach — Get First Reply"`

3. Compute `stepDisplay` from `getStepDisplay(currentStepNumber)`
4. Render a `<Card>` wrapping a `<CardContent className="py-4">` containing:

     - Current Step: `label="Current Step"`, `value={stepDisplay}`, no ragStatus (renders without RAG dot), `ariaLabel="Current roadmap step: {stepDisplay}"`
