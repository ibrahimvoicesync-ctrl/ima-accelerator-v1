**POST /api/reports route.ts** — Port from reference-old/src/app/api/reports/route.ts with V1 import path adjustments:
- Imports: `NextRequest, NextResponse` from "next/server", `z` from "zod", `createClient` from "@/lib/supabase/server", `createAdminClient` from "@/lib/supabase/admin", `VALIDATION` from "@/lib/config", `isValidDateString` from "@/lib/utils"
- Zod `postSchema`:
  ```typescript
  const postSchema = z.object({
    date: z.string().refine(isValidDateString, "Invalid date format (YYYY-MM-DD)"),
    hours_worked: z.number().min(0).max(24),
    star_rating: z.number().int().min(VALIDATION.starRating.min).max(VALIDATION.starRating.max),
    outreach_count: z.number().int().min(VALIDATION.outreachCount.min).max(VALIDATION.outreachCount.max),
    wins: z.string().max(VALIDATION.reportWins.max).optional(),
    improvements: z.string().max(VALIDATION.reportImprovements.max).optional(),
  });
  ```
- POST handler flow:
  1. Auth check: `const supabase = await createClient()`, `supabase.auth.getUser()`, return 401 if no authUser
  2. Admin client: `const admin = createAdminClient()`, fetch profile by `auth_id`, return 404 if no profile
  3. Role check: `profile.role !== "student"` returns 403
  4. JSON parse in try/catch: return 400 on invalid JSON
  5. `postSchema.safeParse(body)`: return 400 with first error message on failure
  6. Check existing: `admin.from("daily_reports").select("id").eq("student_id", profile.id).eq("date", date).maybeSingle()`
  7. If existing: UPDATE with `hours_worked, star_rating, outreach_count, wins: wins || null, improvements: improvements || null, submitted_at: new Date().toISOString()` on `eq("id", existing.id)`, return 200 with data
  8. If not existing: INSERT with `student_id: profile.id, date, hours_worked, star_rating, outreach_count, wins: wins || null, improvements: improvements || null, submitted_at: new Date().toISOString()`, return 201 with data
  9. All DB errors return 500 with error.message
