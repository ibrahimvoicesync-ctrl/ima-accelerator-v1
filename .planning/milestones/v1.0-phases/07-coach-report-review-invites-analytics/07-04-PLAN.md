```tsx
// Check if email already belongs to a registered user
const { data: existingUser } = await admin
  .from("users")
  .select("id")
  .eq("email", parsed.data.email)
  .maybeSingle();

if (existingUser) {
  return NextResponse.json(
    { error: "A user with this email is already registered" },
    { status: 409 }
  );
}
```

This queries the public `users` table (not auth.users) using the already-available `admin` client. If a user with that email exists, it returns 409 Conflict before any invite code is generated or any row is inserted. The admin client is already initialized on line 18, so no new imports or clients are needed.
